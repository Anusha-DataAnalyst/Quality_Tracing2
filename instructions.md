# AI Traceability Instructions

## Goal
Generate traceable code using AI assistance.

## Rule 1
Every code block must include:
- FR or NFR ID
- Traceable method or endpoint

## Rule 2
Use this format:

## FR-01
Description:
Code:
Trace:

## NFR-01
Description:
Code:
Trace:

## Rule 3
Do NOT generate code without linking it to a requirement.

## Rule 4
Trace links must map to real functions in iTrust2.
